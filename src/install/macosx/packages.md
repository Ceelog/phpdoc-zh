使用软件包
----------

Mac OS X 下有几个预编译和打包的 PHP 版本。可以通过它使用标准的配置的
PHP，但是若需要不同的功能集（比如一个安全服务器，或者一个不同的数据库驱动程序），可能需要自己编译
PHP 和／或 web
服务器。如果不熟悉如何自己编译软件，可以试着找一下包含自己所需功能的 PHP
预编译版本。
