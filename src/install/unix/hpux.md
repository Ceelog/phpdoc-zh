针对 HP-UX 平台的安装提示
-------------------------

本节包括在 HP-UX 平台下安装 PHP 的说明和提示。

在 HP-UX 系统下安装 PHP 有两个选择，一是编译，二是安装预编译的可执行包。

官方发布的预编译包可在此地址搜索：<a href="http://software.hp.com/" class="link external">» http://software.hp.com/</a>。
