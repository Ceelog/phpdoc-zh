新增的类常量
------------

5.3.0 中新增了若干类常量:

<a href="/book/pdo.html#Firebird%20(PDO)" class="link">PDO_FIREBIRD</a>:

-   <span class="simpara"> **`PDO::FB_ATTR_DATE_FORMAT`** -
    为日期设置格式. </span>
-   <span class="simpara"> **`PDO::FB_ATTR_TIME_FORMAT`** -
    为时间设置格式. </span>
-   <span class="simpara"> **`PDO::FB_ATTR_TIMESTAMP_FORMAT`** -
    为时间戳设置格式. </span>
