需求
----

构建此扩展不需要其他扩展。

安装
----

使用这些函数不需要安装，它们是 PHP 核心的一部分。

运行时配置
----------

此扩展没有在 `php.ini` 中定义配置指令。

资源类型
--------

本扩展定义了 *process* 资源， <span class="function">proc\_open</span>
函数会返回此资源。
