需求
----

无额外需求，因为标准 C 库中的函数总是可用的。

安装
----

可以使用 **--disable-ctype** 配置项来禁用 ctype 扩展。

PHP 的 Windows
版本已内建对此扩展的支持。不需要载入额外的扩展来使用这些函数。

运行时配置
----------

此扩展没有在 `php.ini` 中定义配置指令。

资源类型
--------

此扩展没有定义资源类型。
