需求
----

构建此扩展不需要其他扩展。

安装
----

要使用本类函数，需要在编译 PHP 时加上 **--enable-calendar**。

PHP 的 Windows
版本已内建对此扩展的支持。不需要载入额外的扩展来使用这些函数。

运行时配置
----------

此扩展没有在 `php.ini` 中定义配置指令。

资源类型
--------

此扩展没有定义资源类型。
