Imagick 是用 ImageMagic API 来创建和修改图像的PHP官方扩展。

ImageMagick® 是用来创建，编辑，合并位图图像的一套组件。
它能够用于读取，转换，写入多种不同格式的图像。 包含 DPX, EXR, GIF, JPEG,
JPEG-2000, PDF, PhotoCD, PNG, Postscript, SVG, 和 TIFF。
